# pdp8

PDP-8 Emulator Project in C++

## General Tasks

* CPU emulator, OMNIBUS
* Peripherals (first and foremost EAE, memory extensions)
* ?Look into realistic timings?
* GUI (Dear ImGUI-docked)

## Expected functionality

* Running programs
* Interactive CLI
* PTTY
* GUI
